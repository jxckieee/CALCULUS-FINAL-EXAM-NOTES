Limits

Definition of a Limit

limx→a f(x) = L means f(x) gets
arbitrarily close to L as x → a

Left-hand: x → a⁻  |  Right-hand: x → a⁺

Limit Exists When

limx→a⁻ f(x) = limx→a⁺ f(x) = L

Both one-sided limits must be equal

Continuity at x = a

1. f(a) is defined
2. limx→a f(x) exists
3. limx→a f(x) = f(a)

Limit Laws

Sumlim[f ± g] = L ± M
Productlim[f · g] = L · M
Quotientlim[f/g] = L/M, M ≠ 0
Powerlim[f]ⁿ = Lⁿ
Constantlim[c·f] = c·L
Rootlim[√f] = √L (L ≥ 0)

Special / Important Limits

limx→0 sin(x)/x = 1
limx→0 (1 - cos x)/x = 0
limx→0 (eˣ - 1)/x = 1
limx→∞ (1 + 1/x)ˣ = e

Indeterminate Forms

0/0 ∞/∞ 0·∞ ∞ - ∞ 0⁰ ∞⁰ 1^∞

→ Use L'Hôpital's Rule or algebraic manipulation

L'Hôpital's Rule

If 0/0 or ±∞/∞ form:
lim f(x)/g(x) = lim f′(x)/g′(x)

Differentiate numerator & denominator separately. May apply repeatedly.

Techniques for Evaluating Limits

  • Direct substitution — plug in x = a first
  • Factor & cancel common factors
  • Multiply by conjugate (for radicals)
  • Divide by highest power of x (for ∞ limits)
  • L'Hôpital's Rule for indeterminate forms
  • Squeeze Theorem when bounded by known limits

Squeeze Theorem

If g(x) ≤ f(x) ≤ h(x) near a, and
lim g(x) = lim h(x) = L,
then lim f(x) = L

Infinite Limits (Asymptotes)

If lim f(x) = ±∞ → vertical asymptote at x = a
If limx→±∞ f(x) = L → horizontal asymptote y = L

Derivatives — Fundamentals

Definition of the Derivative

f′(x) = limh→0 [f(x+h) − f(x)] / h

Also: f′(a) = limx→a [f(x)−f(a)] / (x−a)

Differentiability

If f is differentiable at a → f is continuous at a
Converse is FALSE — continuous ≠ differentiable

Fails at: corners, cusps, vertical tangents, discontinuities

Notation

f′(x) = y′ = dy/dx = df/dx = D[f]

Higher order: f″, f‴, f⁽⁴⁾ or d²y/dx², d³y/dx³ ...

Basic Derivative Rules

Constantd/dx [c] = 0
Identityd/dx [x] = 1
Power Ruled/dx [xⁿ] = nxⁿ⁻¹
Constant multd/dx [cf] = c·f′
Sum/Diffd/dx [f ± g] = f′ ± g′
Exponentiald/dx [eˣ] = eˣ
General expd/dx [aˣ] = aˣ ln(a)
Natural logd/dx [ln x] = 1/x
Log base ad/dx [logax] = 1/(x ln a)
Derivative Rules — Product, Quotient, Chain

Product Rule

d/dx [f · g] = f′g + fg′

Mnemonic: "first times derivative of second, plus second times derivative of first"

Quotient Rule

d/dx [f/g] = (f′g − fg′) / g²

Mnemonic: "low d-high minus high d-low, over low squared"  →  (lo·dhi − hi·dlo) / lo²

Chain Rule

d/dx [f(g(x))] = f′(g(x)) · g′(x)

Derivative of outer function (keeping inside intact) times derivative of inner function

Chain Rule — Examples

sin(x²)cos(x²) · 2x
e^(3x)e^(3x) · 3
(x²+1)⁵5(x²+1)⁴ · 2x
ln(sin x)(1/sin x) · cos x = cot x
√(x³+1)(3x²) / (2√(x³+1))

Implicit Differentiation

  • Differentiate both sides w.r.t. x
  • Apply chain rule: d/dx[f(y)] = f′(y)·(dy/dx)
  • Collect all dy/dx terms on one side
  • Factor out and solve for dy/dx

Example

x² + y² = 25 → 2x + 2y(dy/dx) = 0 → dy/dx = −x/y

Logarithmic Differentiation

  • Take ln of both sides: ln y = ln[f(x)]
  • Simplify using log properties
  • Differentiate implicitly
  • Multiply both sides by y, substitute back

Use for: yˣ, xˣ, products of many functions

Related Rates Strategy

  • Identify all quantities and rates
  • Write an equation relating variables
  • Differentiate implicitly with respect to t
  • Plug in known values and solve

Common mistake

Do NOT substitute known values before differentiating — only after.

Fundamental Theorem of Calculus

FTC Part 1 — Differentiation of an Integral

If F(x) = ∫ax f(t) dt, then F′(x) = f(x)

The derivative "undoes" the integral. f must be continuous on [a, b].

FTC Part 1 — Chain Rule Extension

d/dx ∫ag(x) f(t) dt = f(g(x)) · g′(x)

Apply chain rule when the upper limit is a function of x

FTC Part 2 — Evaluation Theorem

∫ab f(x) dx = F(b) − F(a), where F′ = f

Find antiderivative F, evaluate at endpoints, subtract. This is the workhorse of definite integrals.

Definite Integral Properties

Reverse limits∫ba f dx = −∫ab f dx
Same limits∫aa f dx = 0
Split interval∫ab f = ∫ac f + ∫cb f
Linearity∫[cf ± dg] = c∫f ± d∫g
Comparisonf ≥ g on [a,b] → ∫f ≥ ∫g
Even fn∫−aa f dx = 2∫0a f dx
Odd fn∫−aa f dx = 0

Net Change Theorem

∫ab F′(x) dx = F(b) − F(a)

The integral of a rate of change = net change in the quantity

Total vs Net Displacement

Net displacement: ∫v(t)dt
Total distance: ∫|v(t)|dt (split where v = 0)

Integration Techniques

U-Substitution (Sub Rule)

∫f(g(x))·g′(x) dx = ∫f(u) du
where u = g(x), du = g′(x) dx

U-Sub Steps

  • Choose u = g(x) — usually the "inside" function
  • Compute du = g′(x) dx
  • Solve for dx: dx = du/g′(x)
  • Substitute everything, integrate in terms of u
  • Back-substitute u = g(x)
  • For definite integrals: change limits in terms of u

U-Sub — Common Choices

∫sin(x²)·2x dxu = x²
∫eˣ·eˣ dxu = eˣ
∫(x²+1)³·2x dxu = x²+1
∫cos x/sin x dxu = sin x
∫x·e^(x²) dxu = x²
∫ln(x)/x dxu = ln x

Tip

What you pick as u must have its derivative (up to a constant) also present in the integrand.

1st & 2nd Derivative Tests

First Derivative Test

At critical point c:
f′: + → − ⟹ local max
f′: − → + ⟹ local min
No sign change ⟹ neither

Second Derivative Test

At critical point c where f′(c) = 0:
f″(c) > 0 ⟹ local min
f″(c) < 0 ⟹ local max
f″(c) = 0 ⟹ inconclusive

Area Between Curves

A = ∫ab [f(x) − g(x)] dx

where f(x) ≥ g(x) on [a, b]

  • Find intersection points (set f = g)
  • Determine which function is on top
  • Integrate (top − bottom) over interval
  • If they cross, split into sub-intervals

Tangent Line & Linear Approx

Tangent Line at (a, f(a))

y − f(a) = f′(a)(x − a)

Linear Approximation

L(x) = f(a) + f′(a)(x − a)

Good approximation for x near a

Differentials

dy = f′(x) dx

Δy ≈ dy for small Δx

Average Value of a Function

favg = (1/(b−a)) · ∫ab f(x) dx

Key Theorems

Intermediate Value Theorem (IVT)

If f is continuous on [a,b] and N is between f(a) and f(b), then ∃ c ∈ (a,b) such that f(c) = N

Use to show a root exists: if f(a) and f(b) have opposite signs, there's a zero in (a,b)

Mean Value Theorem (MVT)

If f is continuous on [a,b] and differentiable on (a,b), then ∃ c ∈ (a,b) such that:
f′(c) = [f(b) − f(a)] / (b − a)

The instantaneous rate of change equals the average rate of change at some point

Rolle's Theorem

If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then ∃ c ∈ (a,b) such that f′(c) = 0

Special case of MVT — a horizontal tangent must exist between equal values

Extreme Value Theorem (EVT)

If f is continuous on [a,b], then f attains an absolute maximum and minimum on [a,b]

To find them: evaluate f at all critical points AND endpoints

Finding Absolute Extrema on [a, b]

  • Find all critical points (where f′ = 0 or f′ is undefined) in (a,b)
  • Evaluate f at each critical point
  • Evaluate f at the endpoints: f(a) and f(b)
  • The largest value is the absolute max; the smallest is the absolute min

Common Mistakes

Derivative

d/dx[uv] ≠ u′v′ — must use product rule

Chain Rule

d/dx[f(g(x))] ≠ f′(g(x)) — must multiply by g′(x)

Integration

∫1/x² dx ≠ ln|x²| — apply power rule: −1/x + C

Signs of f, f′, f″

f′ > 0increasing
f′ = 0critical point
f″ > 0concave up (∪)
f″ = 0inflection pt candidate
f″ < 0concave down (∩)
