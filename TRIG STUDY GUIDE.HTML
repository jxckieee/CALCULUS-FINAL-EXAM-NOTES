The Six Trigonometric Functions

1.1 Right Triangle Definitions

In a right triangle with acute angle θ, label the three sides: Opposite (across from θ), Adjacent (next to θ, not the hypotenuse), Hypotenuse (longest side, across from 90°).

FunctionAbbrev.RatioDescription
Sinesin θOPP / HYPOpposite over Hypotenuse
Cosinecos θADJ / HYPAdjacent over Hypotenuse
Tangenttan θOPP / ADJOpposite over Adjacent
Cosecantcsc θHYP / OPPReciprocal of sin
Secantsec θHYP / ADJReciprocal of cos
Cotangentcot θADJ / OPPReciprocal of tan
Memory Trick — SOH CAH TOA
SOH → Sin = Opp/Hyp
CAH → Cos = Adj/Hyp
TOA → Tan = Opp/Adj

For reciprocals: CHO SHA CAO — Csc = H/O, Sec = H/A, Cot = A/O. Or simply flip SOH CAH TOA.

1.2 Unit Circle Definitions

For any angle θ whose terminal side hits the unit circle at point (x, y):

Sine
sin θ = y
Cosine
cos θ = x
Tangent
tan θ = y/x  (x ≠ 0)
Cosecant
csc θ = 1/y  (y ≠ 0)
Secant
sec θ = 1/x  (x ≠ 0)
Cotangent
cot θ = x/y  (y ≠ 0)

1.3 Reciprocal & Ratio Identities

Reciprocals
sin θ · csc θ = 1
cos θ · sec θ = 1
tan θ · cot θ = 1
From reciprocals
csc θ = 1/sin θ
sec θ = 1/cos θ
cot θ = 1/tan θ
Ratio identities
tan θ = sin θ / cos θ
cot θ = cos θ / sin θ

2.1 What It Is

A circle with radius = 1 centered at the origin. Every point has coordinates (cos θ, sin θ) — the foundation of all trigonometry.

2.3 How to Memorize the Unit Circle

Learn the 4 axis points: (1,0) · (0,1) · (−1,0) · (0,−1)
Memorize Q1 sin values (0°→90°): 0, 1/2, √2/2, √3/2, 1
Cosine is sin in reverse: 1, √3/2, √2/2, 1/2, 0
Use reference angles for other quadrants — same magnitude, sign from quadrant rule.
The √3, √2, 1 Pattern

Numerators in Q1 are just √3, √2, 1 (and reverses), all over 2. You only need to memorize one row — the other is the reverse!

Reference Angle Rules

Q2: ref = 180° − θ → sin stays +, cos −
Q3: ref = θ − 180° → sin −, cos −
Q4: ref = 360° − θ → sin −, cos stays +

2.4 Degrees ↔ Radians Conversion

Degrees → Radians
radians = degrees × (π / 180)
Radians → Degrees
degrees = radians × (180 / π)
Quick Conversions

π = 180° · π/2 = 90° · π/3 = 60° · π/4 = 45° · π/6 = 30°
2π = 360° · 3π/2 = 270° · 2π/3 = 120° · 3π/4 = 135° · 5π/6 = 150°

3.2 Key Shape Notes

FunctionBehavior
sin xStarts at (0,0) → rises to 1 at π/2 → back to 0 at π → down to −1 at 3π/2 → back to 0 at 2π

3.3 General Transformation Formula

Formula

y = A · f(Bx − C) + D

ParameterNameEffect
AAmplitudeVertical stretch/compress; |A| > 1 → taller; 0 < |A| < 1 → shorter; A < 0 → reflects over x-axis
BFrequencyHorizontal stretch/compress; New Period = (Original Period) / |B|
CPhase ShiftHorizontal translation; Phase Shift = C/B; positive → shifts RIGHT; negative → shifts LEFT
DVertical ShiftMoves midline up (D > 0) or down (D < 0); new midline is y = D

3.4 Graphing a Transformed Function — Step by Step

Identify A, B, C, D from y = A·f(Bx − C) + D
Find new period = Original Period / |B|
Find phase shift = C / B (your new starting x)
Find midline: y = D
Find max = D + |A| and min = D − |A|
Divide one period into 4 equal intervals (5 key points for sin/cos)
Apply reflection if A < 0, then sketch
Worked Example — y = −3 sin(2x − π) + 1

A = −3, B = 2, C = π, D = 1
Amplitude = |−3| = 3 · Reflection: yes
Period = 2π / 2 = π · Phase Shift = π/2 to the right
Midline: y = 1 · Max: y = 4 · Min: y = −2
Key x-values: π/2, 3π/4, π, 5π/4, 3π/2

04

Asymptotes

4.1 Why Asymptotes Occur

Vertical asymptotes appear wherever a trig function is undefined — where its denominator equals zero. sin x and cos x have no vertical asymptotes (defined for all reals).

4.2 Asymptote Formulas

FunctionVertical AsymptotesReason
tan x = sin x / cos xx = π/2 + nπcos x = 0
csc x = 1 / sin xx = nπsin x = 0
sec x = 1 / cos xx = π/2 + nπcos x = 0
cot x = cos x / sin xx = nπsin x = 0

4.3 Asymptotes After Transformations

For y = A·tan(Bx − C) + D, set the argument equal to the critical value and solve for x.

For tan/sec
Set Bx − C = π/2 + nπ
→ solve for x
For csc/cot
Set Bx − C = nπ
→ solve for x
Example — Asymptotes of y = tan(2x − π)

Set 2x − π = π/2 + nπ → 2x = 3π/2 + nπ → x = 3π/4 + nπ/2
Asymptotes at x = 3π/4, 5π/4, 7π/4, … (every π/2 apart)

4.4 Finding Asymptotes — Checklist

Identify which function you have (tan, csc, sec, or cot)
Write the standard asymptote formula from the table above
Replace x with your full argument (e.g., 3x + π/4)
Set equal to the critical value and solve for x
Spacing between consecutive asymptotes = period of the function
05

All Essential Formulas

5.1 Pythagorean Identities

Fundamental
sin²θ + cos²θ = 1
Divide by cos²θ
1 + tan²θ = sec²θ
Divide by sin²θ
1 + cot²θ = csc²θ
Shortcut

Only memorize sin²θ + cos²θ = 1. Derive the others by dividing through. Useful rearrangements: sin²θ = 1 − cos²θ and cos²θ = 1 − sin²θ.

5.2 Co-function Identities

Sine / Cosine
sin θ = cos(90° − θ)
cos θ = sin(90° − θ)
Tan / Cot
tan θ = cot(90° − θ)
cot θ = tan(90° − θ)
Sec / Csc
sec θ = csc(90° − θ)
csc θ = sec(90° − θ)

5.3 Even / Odd Identities

Even (cos, sec)
cos(−θ) = cos θ
sec(−θ) = sec θ
Odd (sin, tan, csc, cot)
sin(−θ) = −sin θ
tan(−θ) = −tan θ
csc(−θ) = −csc θ
cot(−θ) = −cot θ
Memory Hook

Only Cos and Sec are EVEN (C comes before S in alphabet). All 4 others are ODD.

5.4 Sum & Difference Formulas

Sine
sin(A ± B) = sin A cos B ± cos A sin B
Cosine (signs flip!)
cos(A ± B) = cos A cos B ∓ sin A sin B
Tangent
tan(A ± B) = (tan A ± tan B) / (1 ∓ tan A tan B)
Memory Hook

SIN: same sign pattern. COS: signs FLIP. TAN: top uses given sign, bottom uses opposite sign.

5.5 Double Angle Formulas

Sine
sin(2θ) = 2 sin θ cos θ
Cosine (3 forms)
cos(2θ) = cos²θ − sin²θ
= 2cos²θ − 1
= 1 − 2sin²θ
Tangent
tan(2θ) = 2 tan θ / (1 − tan²θ)

5.6 Half Angle Formulas

Sine
sin(θ/2) = ± √[(1 − cos θ) / 2]
Cosine
cos(θ/2) = ± √[(1 + cos θ) / 2]
Tangent (3 forms)
tan(θ/2) = ± √[(1−cosθ)/(1+cosθ)]
= sin θ / (1 + cos θ)
= (1 − cos θ) / sin θ
Note

The ± depends on which quadrant θ/2 lies in.

5.7 Product-to-Sum Formulas

sin · sin
sin A sin B = ½[cos(A−B) − cos(A+B)]
cos · cos
cos A cos B = ½[cos(A−B) + cos(A+B)]
sin · cos
sin A cos B = ½[sin(A+B) + sin(A−B)]

5.8 Sum-to-Product Formulas

sin A + sin B
2 sin[(A+B)/2] cos[(A−B)/2]
sin A − sin B
2 cos[(A+B)/2] sin[(A−B)/2]
cos A + cos B
2 cos[(A+B)/2] cos[(A−B)/2]
cos A − cos B
−2 sin[(A+B)/2] sin[(A−B)/2]

5.9 Power-Reducing Formulas

sin²θ
(1 − cos 2θ) / 2
cos²θ
(1 + cos 2θ) / 2
tan²θ
(1 − cos 2θ) / (1 + cos 2θ)

5.10 Law of Sines & Law of Cosines

For non-right triangles with sides a, b, c opposite to angles A, B, C:

Law of Sines
a/sin A = b/sin B = c/sin C
Law of Cosines
c² = a² + b² − 2ab cos C
a² = b² + c² − 2bc cos A
When to Use Which

Law of Sines: use when you know an angle AND the side opposite it.
Law of Cosines: use for SSS (all 3 sides) or SAS (2 sides + included angle).

Inverse Trigonometric Functions

6.1 What They Do

Inverse trig functions answer: "What angle gives this trig value?" Written as sin⁻¹(x) or arcsin(x).

FunctionDomainRangeReads as…
arcsin / sin⁻¹[−1, 1][−π/2, π/2] (Q1 & Q4)The angle whose sine is x
arccos / cos⁻¹[−1, 1][0, π] (Q1 & Q2)The angle whose cosine is x
arctan / tan⁻¹(−∞, ∞)(−π/2, π/2) (Q1 & Q4)The angle whose tangent is x
arccsc / csc⁻¹(−∞,−1]∪[1,∞)[−π/2,0)∪(0,π/2]The angle whose cosecant is x
arcsec / sec⁻¹(−∞,−1]∪[1,∞)[0,π/2)∪(π/2,π]The angle whose secant is x
arccot / cot⁻¹(−∞, ∞)(0, π)The angle whose cotangent is x
Critical Warning

sin⁻¹(x) ≠ 1/sin(x)
sin⁻¹(x) means arcsine — the inverse function. It does NOT mean (sin x)⁻¹ = csc x. Always read it as "the angle whose sine is x."

7.2 The Trig Identity Hierarchy

1.

Convert everything to sin and cos first

2.

Apply Pythagorean identities to reduce sin² + cos² groups

3.

Factor or combine fractions if needed

4.

Look for double/half angle opportunities if the argument changes

5.

Work on the MORE COMPLEX side of an equation first

7.3 Essential Shift Relationships

Shift by +π/2
sin(θ + π/2) = cos θ
cos(θ + π/2) = −sin θ
Shift by −π/2
sin(θ − π/2) = −cos θ
cos(θ − π/2) = sin θ

7.4 Common Mistakes to Avoid

  • tan 90° is UNDEFINED — not infinity. Write "undefined" on tests.
  • sin⁻¹(x) is NOT 1/sin(x). It is the arcsine (inverse function).
  • Period formula: new period = original period / |B|, NOT multiplied by B.
  • Phase shift = C/B (not just C). Always divide by B.
  • Amplitude is |A|, never negative. The − in A causes reflection, not negative amplitude.
  • cos(A−B) ≠ cos A − cos B. Trig functions are NOT distributive over addition.
  • sin²θ means (sin θ)², not sin(θ²). The square wraps the whole function value.

7.5 Final Study Tips

Draw the unit circle from memory daily for 2 weeks — fill in all 16 key angles.
Derive SOH-CAH-TOA and the Pythagorean identity from scratch each time — don't just stare at them.
Practice graphing y = A·sin(Bx−C)+D with random A/B/C/D values.
For proofs, never touch both sides at once — only manipulate ONE side.
Use Anki or flashcards for the exact formula phrasing of sum/difference formulas.
